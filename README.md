# Uniform Tests Research

This project contains Python scripts for performing uniformity tests on datasets. 
It includes functions to load data, process it, analyze results, and save outputs.

## Functions
- load_data(): Reads input data into the program
- process_data(): Cleans and prepares data for analysis
- analyze_results(): Performs statistical tests and calculations
- save_output(): Saves results to a file

## Usage
Run the tests using pytest in the terminal.
